# FileNest

FileNest is a simple web-based Document Management System developed using HTML, CSS, and JavaScript. It allows users to upload, manage, search, share, and organize files through an easy-to-use interface inspired by cloud storage platforms like Google Drive and DigiLocker.


## Features

- User Registration and Login
- Dashboard for file management
- Upload and manage files
- Search functionality
- File sharing permissions
- Move files to Trash
- View file details
- Local storage based data management
- Responsive and modern UI

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage API
